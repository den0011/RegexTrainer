## Regex Trainer — Тренажёр регулярных выражений на Qt/C++

### Описание:

Простой и наглядный тренажёр для изучения и практики регулярных выражений (regex), написанный на C++ с использованием Qt Widgets. Приложение позволяет вводить регулярное выражение и тестовую строку, подсвечивает совпадения, отображает список найденных совпадений и показывает подсказки по выражению.

### Особенности:

* Поддержка флагов Case Insensitive и Multiline.
* Подсветка совпадений в реальном времени.
* Список найденных совпадений для наглядного анализа.
* Встроенные примеры: email, телефоны, IP, даты, HTML-теги, числа, слова с заглавной буквы и другие.
* Подсказки по регулярным выражениям и типичные ошибки.
* Простая структура проекта, полностью на Qt/C++, без сторонних библиотек.
